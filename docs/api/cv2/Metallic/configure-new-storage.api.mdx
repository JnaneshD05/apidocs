## Configure New Storage



This API adds a cloud storage of type "Metallic Recovery Reserve" or "Microsoft Azure Storage".

#### Request Body

| **Field** | **Description** | **Type** | **Required** | **Notes** |
| --- | --- | --- | --- | --- |
| type | type of cloud storage | string | True | Available types - \[**Metallic**, **Azure**\] |
| name | storage name | string | True | Field is valid for type **Azure** only |
| region | region id or name where storage will be created | integer (id) , string (name) | True |  |
| storageClass | azure storage class | string | True | Field is valid for type **Metallic** only. Available types - \[**MRR_HOT**, **MRR_COOL**\] |
| credential | cloud credential id or name | integer (id) , string (name) | True | Field is valid for type **Azure** only |
| container | container or library name | string | True | Field is valid for type **Azure** only |
| backupGateways | backup gateway name or id | integer (id) , string (name) | False | Field is valid for type **Azure** only |

#### Response Body

| **Field** | **Description** | **Type** | **Notes** |
| --- | --- | --- | --- |
| id | newly created cloud storage id | integer |  |
| name | newly created cloud storage name | string |  |
