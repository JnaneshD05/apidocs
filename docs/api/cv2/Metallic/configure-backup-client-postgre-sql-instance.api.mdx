## Configure Backup Client PostgreSQL Instance



This API creates Cloud Database Instance for PostgreSQL.

All the databases in the PostgreSQL instance will be backed up after creation.

Use GET azuredb/postgresql/instance API to get the dbInstanceIdentifier and port.

#### Request Body

| **Field** | **Description** | **Type** | **Required** | **Notes** |
| --- | --- | --- | --- | --- |
| region | region id | integer | True |  |
| cloudAccount | cloud account id or name | integer (id) , string (name) | True |  |
| dbInstanceIdentifier | database instance identifier | string | True | Use GET azuredb/postgresql/instance API to get the dbInstanceIdentifier and port |
| database.username | database username | string | True |  |
| database.password | database password | string | True |  |
| endPoint | endpoint to connect to the database instance | string | True |  |
| port | port to connect to the database instance | integer | True |  |
| SSL_CA_Certificate | SSL CA file | string | False | used when SSL is enabled |
| backupGateways | backup gateway id or name | integer (id) , string (name) | True |  |
| plan | plan id or name | integer (id) , string (name) | True |  |

#### Response Body

| **Field** | **Description** | **Type** | **Notes** |
| --- | --- | --- | --- |
| id | newly created instance id | integer |  |
| name | newly created instance name | string |  |
