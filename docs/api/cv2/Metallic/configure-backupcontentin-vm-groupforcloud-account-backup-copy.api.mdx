## Configure Backup content in VMGroup for cloud Account Backup Copy



This API creates a vm group for Cloud account on metallic azure.

Content is optional . If not provided all VMs in the account are backed up. Any VMs tagged to the particular tag in the same region as plan will be backed up

#### Request Body

| **Field** | **Description** | **Type** | **Required** | **Notes** |
| --- | --- | --- | --- | --- |
| cloudAccount | cloud account name or id | integer (id) , string (name) | True |  |
| plan | plan name or id | integer (id) , string (name) | True |  |
| content | virtual machine content to be backed up | array | True | Payload contains the Array of the contents of the Cosmos DB that should be explicitly backed up. We get this from cloudDBContentPreview API for COSMOS DB. |

#### Response Body

| **Field** | **Description** | **Type** | **Notes** |
| --- | --- | --- | --- |
| subclientId | newly created Vmgroup  id | integer |  |
