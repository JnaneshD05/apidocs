## Configure New Storage1



This API adds a cloud storage of type "Metallic Cloud Storage" or "Microsoft Azure Storage".

#### Request Body

| **Field** | **Description** | **Type** | **Required** | **Notes** |
| --- | --- | --- | --- | --- |
| type | type of cloud storage | string | True | available types - \[Metallic, Azure\] |
| name | storage name | string | False |  |
| region | region id or name where storage will be created | integer (id) , string (name) | True |  |
| storageClass | azure storage class | string | False | available types - \[MRR_HOT, MRR_COOL\] |
| credential | cloud credential id or name | integer (id) , string (name) | False |  |
| container | container or library name | string | False |  |
| backupGateways | backup gateway name or id | integer (id) , string (name) | False |  |
