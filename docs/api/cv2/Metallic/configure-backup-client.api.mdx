## Configure Backup Client



This API creates Cloud Database Instance for SQL Server.

All the databases in the SQL Server instance will be backed up after creation.

#### Request Body

| **Field** | **Description** | **Type** | **Required** | **Notes** |
| --- | --- | --- | --- | --- |
| region | region where instance needs to be created | integer (id) , string (name) | True |  |
| instanceName | MSSQL server instance name | string | True |  |
| connectionString | MSSQL connection string | string | True |  |
| credential | credential name or id | integer (id) , string (name) | True |  |
| cloudAccount | cloud account id or name | integer (id) , string (name) | True |  |
| backupGateways | backup gateway id or name | integer (id) , string (name) | True |  |
| plan | plan id or name | integer (id) , string (name) | True |  |

#### Response Body

| **Field** | **Description** | **Type** | **Notes** |
| --- | --- | --- | --- |
| id | newly created instance id | integer |  |
| name | newly created instance name | string |  |
