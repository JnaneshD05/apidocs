## VM Out Place Recover



This operation recovers a virtual machine and returns details for the restore job.


Note:
<i>
 vmClientGUID
</i>
is the client GUID of the backed-up VM. You can use [_GET Virtualization Clients_](https://api.commvault.com/?version=latest#e12f8a73-87cb-d7d3-872d-82cbeadd5c4a) to fetch the client GUID from the client properties of the VM.
